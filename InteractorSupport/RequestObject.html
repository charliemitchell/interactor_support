Overview

A base module for building validated, transformable, and optionally nested request objects.

It builds on top of ActiveModel::Model, adds coercion, default values, attribute transforms, and automatic context conversion (via #to_context). It integrates tightly with InteractorSupport::Configuration to control return behavior and key formatting.

Examples:

Simple usage

class CreateUserRequest
  include InteractorSupport::RequestObject

  attribute :name, transform: [:strip, :downcase]
  attribute :email
  attribute :metadata, default: {}
end

CreateUserRequest.new(name: " JOHN ", email: "hi@example.com")
# => { name: "john", email: "hi@example.com", metadata: {} }
